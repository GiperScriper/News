<h2>Агрегатор новостей (ver. 0.1)</h2>

<h3>Используемые технологии:</h3>
<ul>
	<li>Python</li>
	<li>Django</li>
	<li>Django Rest Framework</li>
	<li>SQLite3</li>
	<li>HTML, CSS (Twitter Bootstrap)</li>
	<li>JS (JQuery)</li>
</ul>

<h3>Функционал:</h3>
<ul>
	<li>Авторизация</li>
	<li>Добавление новостей</li>
	<li>Admin interface with Django Suit</li>
	<li>Лайк новостей</li>
	<li>RESTful API</li>
</ul>
