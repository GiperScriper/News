<h2>News aggregation (ver. 0.1)</h2>

<h3>Technologies:</h3>
<ul>
	<li>Python</li>
	<li>Django</li>
	<li>Django Rest Framework</li>
	<li>SQLite3</li>
	<li>HTML, CSS (Twitter Bootstrap)</li>

<h3>functionality:</h3>
<ul>
	<li>auth</li>
	<li>adding news</li>
